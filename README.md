# Hardened Node.js App - ISP Dashboard
## Features
 - Authentication with JWT Tokens
 - Password hashing with salt
 - SQLite (In memory)
 - CSRF
 - Jade template engine
 - Sessions stored in DB

## Installation
```
yarn install && node index.mjs
```
